# User Directory App

A modern Android User Directory App built with **Jetpack Compose**, **Room Database**, **Retrofit** in **Kotlin**.

The app fetches user data from a remote API and displays it in a clean, modern Compose UI.

CPSC 411A - Project (Fall 2025)

---

## Features

- Fetch users from a REST API using Retrofit + OkHttp
- Local caching with Room Database
- Reactive UI with Jetpack Compose
- MVVM Architecture with ViewModel + Repository
- Kotlin Coroutines + Flows for async operations
- Material 3 UI components
